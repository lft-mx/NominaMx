# Calculadora Laboral México 🇲🇽

Calculadora de sueldo neto con ISR, IMSS, Infonavit, Fondo de ahorro y Vales de despensa.  
Datos actualizados SAT 2026.

## Características
- Cálculo de ISR con subsidio (método real mensual + subsidio)
- Comparación con método directo
- Gráfica de porcentajes
- Modo oscuro
- Bilingüe (Español/Inglés)
- Espacios para publicidad (Google AdSense)
- Disclaimer legal
